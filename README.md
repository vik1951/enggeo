# EnGeo

Цей пакет призначений для збору, зберігання та обробки матеріалів з інженерно-геологічних досліджень.
Працює у вертуальному середовищи з використанням:

    Python 3.10+
    PostgreSQL 14+
    JasperStudio 20+
    OpenJDK-8
    PyQt5 5.15+
    numpy 2.2+
    pyreportjasper 1.0.2
    psycorg2 2.9+
